**Handwritten Digit Classification**

This project trains a neural network to classify handwritten digits using the MNIST dataset.

*Overview*
Uses the MNIST dataset (28×28 grayscale images of digits 0–9)
Trains a simple neural network with TensorFlow
Evaluates performance and shows predictions
Includes a clean src/-based project structure and basic tests
Installation

Using uv (recommended):

uv sync

*Usage*

Run the training script:
python main.py

Run Tests
uv run pytest

Requirements
Python ≥ 3.11
numpy
matplotlib
tensorflow
scikit-learn
